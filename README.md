# 2025 C# Programming & .NET Development Archive

This repository is a dedicated space for storing and backing up my C# programming coursework and exercises from 2025. It tracks my journey from basic syntax to building functional applications using the .NET ecosystem.

## 🛠 What I Practiced & Learned

### 1. Object-Oriented Programming (OOP)
- **Class & Objects**: Defined real-world entities using classes and managed state through properties and fields.
- **Inheritance & Polymorphism**: Implemented reusable code structures and method overriding to handle diverse data types.
- **Encapsulation**: Used access modifiers (`public`, `private`, `protected`) to ensure data security and integrity.

### 2. C# Core Features
- **Data Handling**: Mastered C# specific data types, collections (List, Dictionary), and arrays for efficient data storage.
- **LINQ**: Practiced Language Integrated Query to filter and manipulate data collections with concise code.
- **Exception Handling**: Built robust applications by implementing `try-catch` blocks to manage runtime errors.

### 3. Application Development with Visual Studio
- **Windows Forms (WinForms)**: Designed GUI-based desktop applications, handling user events like button clicks and text input.
- **File I/O**: Learned to read from and write to external files to persist application data.
- **Debugging**: Utilized Visual Studio's powerful debugging tools to trace code and resolve logical errors.

- **Projects**: Mini-applications (e.g., Management systems, Calculators) built during the semester.
